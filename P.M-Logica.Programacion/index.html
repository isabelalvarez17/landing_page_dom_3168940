Domina la Interacción: Galería y Traducción

Bienvenido a nuestro proyecto de manipulación del DOM (Modelo de Objeto de Documento). Este sitio demuestra cómo JavaScript puede modificar dinámicamente el contenido, las imágenes y la interfaz de usuario para crear experiencias interactivas.

Funcionalidades DOM implementadas: Galería, Lightbox y Traducción.

Detalle del Proyecto

Este modal (o Lightbox) es un elemento que se muestra u oculta mediante la manipulación de clases CSS (en este caso, la clase 'oculto') con JavaScript, cumpliendo con el requisito de mostrar una descripción.
